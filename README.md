# NumPy Data Explorer

This project demonstrates fundamental and advanced NumPy operations performed as part of an internship task.

## 📌 Project Overview
The goal of this task is to explore NumPy capabilities for efficient numerical computation and compare its performance with standard Python lists.

## 🔧 Tools Used
- Python
- NumPy

## 🚀 Features Implemented

### 1️⃣ Array Creation
- Created 1D and 2D NumPy arrays.

### 2️⃣ Indexing and Slicing
- Accessed elements and performed slicing operations.

### 3️⃣ Mathematical Operations
- Element-wise arithmetic operations.
- Axis-wise row and column calculations.

### 4️⃣ Statistical Operations
- Mean
- Median
- Standard Deviation
- Min & Max

### 5️⃣ Reshaping & Broadcasting
- Converted arrays into different shapes.
- Applied broadcasting for vectorized operations.

### 6️⃣ Save & Load Arrays
- Stored arrays using `.npy` format.
- Loaded saved arrays for reuse.

### 7️⃣ Performance Comparison
- Compared execution time of Python lists vs NumPy arrays.
- Demonstrated efficiency of vectorized computation.

## 🎯 Outcome
This project highlights how NumPy improves performance and simplifies numerical data processing in Python.
